> *Ten nations. Ten languages. One cumulative plan for peace — built verse by verse.*



## What Is This?

**One Table** is a multimedia peace anthem and interactive web experience built entirely with AI — lyrics, music, globe visualization, live data dashboard, charity directory, and merch shop. Every verse is performed in the native language of its country, and each one adds another layer to a cumulative global peace plan.

It was built to ask a simple question:

> *If AI is powerful enough to be used as a weapon — why aren't we using it to build something beautiful instead?*

## 🗂️ Features

### 🌍 Globe View
- Real NASA Earth textures (bump map, specular, cloud layer)
- Atmospheric glow shader
- Real starfield background
- Glowing country markers that pulse when their verse plays
- Globe rotates to face each active country
- Auto-advancing lyrics synced to song
- Audio-reactive holographic data panels

### 📡 Live Data Dashboard
- **World population** — live counter (+4.4 people/sec)
- **USGS earthquakes** — real-time API feed, refreshable
- **CO₂ concentration** — current Mauna Loa readings
- **Deforestation counter** — ticking from Jan 1 of current year
- **Hunger, water, conflict, education** — sourced 2024 UN/WHO data
- **Session timer** — every second counts

## 🚀 Getting Started

No build step. No server. Just two files.

```bash
git clone https://github.com/YOUR_USERNAME/one-table.git
cd one-table
```

Place both files in the same directory:
```
one-table/
├── one-table-hologram.html
├── one-table-song.aac
├── screenshot.png          ← take a screenshot of the running app
└── README.md
```

Open `one-table-hologram.html` in any modern browser. Chrome recommended for best WebGL and Web Audio API performance.

> **Note:** The song file must be in the same directory as the HTML for audio to work.

## 🧠 Why We Built This

AI is at a fork in the road.

In January 2026, the US military used an AI model in a raid on Caracas, Venezuela in which 83 people were killed. The Pentagon is pressuring AI companies to strip their safety guardrails — threatening blacklisting under the Defense Production Act.

It's worth noting: **Anthropic has refused to comply**, holding the line against autonomous lethal targeting. We acknowledge and respect that stand.

But the precedent being set is dangerous. Our response is this project.

The same technology being drafted for warfare built this — a multilingual anthem, a spinning globe, real-time data about hunger and earthquakes and the people we share this planet with, and a directory of organizations doing the actual work of peace.

**AI wrote these words. AI composed this music. AI rendered this Earth.**

We believe AI's next chapter should look more like this.
